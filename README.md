## Pre-requisites
- Install Nodejs v6.9+
- Install Apache Maven

## Start developing

### Run frontend
- cd src/main/resources/static
- npm install
- npm run build

### Run backend
- mvn spring-boot:run
- check in browser http://localhost:8090
- login - "admin", password - "password"

## Configuration files
- src/main/resources/static/package.json
- pom.xml
- resources/application.properties
